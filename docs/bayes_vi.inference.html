bayes_vi.inference.inference module¶

Module providing the Inference Base class.

class bayes_vi.inference.inference.Inference(model, dataset)¶

Bases: object

A base class for different Inference algorithms.

An Inference algorithm consists of:

a Bayesian Model and a tf.data.Dataset.

Note: Every subclass has to implement a fit method.

model¶

A Bayesian probabilistic Model.

Type

Model

dataset¶

A tf.data.Dataset consisting of features (if regression model) and targets.

Type

tf.data.Dataset

num_datapoints¶

Number of examples (i.e. datapoints) in the dataset.

Type

int

fit(*args, **kwargs)¶

Fits the Bayesian model to the dataset.

Has to be implemented in any subclass of Inference.

Parameters

  • args (positional arguments) – positional arguments of the fit method.

  • kwargs (keyword arguments) – keyword arguments of the fit method.

Raises

NotImplementedError – If the fit method is not implemented in a subclass.
